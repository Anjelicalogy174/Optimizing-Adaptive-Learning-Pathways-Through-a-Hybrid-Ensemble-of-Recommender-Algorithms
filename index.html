Adaptive Learning Pathway System LIVE

Thesis Demonstration · Anjelica M. Castillo

One intelligent recommendation engine, three user interfaces

Although each interface serves a different purpose, they all interact with the same machine learning pipeline. The Survey collects learner data, the Dashboard presents cohort-level insights, and the Instructor page provides detailed learner analysis. Every recommendation is generated in real time through the same /recommend endpoint, ensuring consistent results across the entire system.
